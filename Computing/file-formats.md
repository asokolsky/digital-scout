# File Formats

Binary vs text-based:

Binary|Text
------|----
Unreadable by human|Readable
Used for data such as audio, video, executables|Used for configuration, data, etc.

* [List of file formats](https://en.wikipedia.org/wiki/List_of_file_formats)
* [Text-based formats](https://en.wikipedia.org/wiki/List_of_file_formats#Text-based)
include [document markup](https://en.wikipedia.org/wiki/List_of_document_markup_languages)
formats, including most important ones
[HTML](https://en.wikipedia.org/wiki/HTML) and
[markdown](https://en.wikipedia.org/wiki/Markdown).

Audio: [wav](https://en.wikipedia.org/wiki/WAV),
[mp3](https://en.wikipedia.org/wiki/MP3).

Video: [mjpg](https://en.wikipedia.org/wiki/Motion_JPEG),
[mkv](https://en.wikipedia.org/w.iki/Matroska).

File formats are subject to fashion:

* out: [xml](https://en.wikipedia.org/wiki/XML), wav or any other
[riff](https://en.wikipedia.org/wiki/Resource_Interchange_File_Format);
* in: [yaml](https://en.wikipedia.org/wiki/YAML),
[json](https://en.wikipedia.org/wiki/JSON),
[ini/toml](https://en.wikipedia.org/wiki/TOML);
* timeless: [csv](https://en.wikipedia.org/wiki/Comma-separated_values).

Read [Markdown Ate The World](https://matduggan.com/markdown-ate-the-world/) on
the history of doc-> docx -> md.
